# 26SummerBlue 版本管理文档

## 当前版本

v0.5.0

## 最近修改

- 新增 `LevelScrollManager`：摄像机沿 X 正方向匀速移动，形成关卡强制卷轴。
- 新增 `EnemySpawnPoint`：摄像机右边缘到达触发 X 坐标时，生成指定敌人或敌人组。
- 背景块默认固定在关卡世界坐标中，不再自行左移；由移动摄像机产生卷轴观感。
- 背景创建菜单改为沿 X 正方向铺设多块静态长条背景。
- `EnemyBase` 增加公开移动速度设置接口，出生点可覆盖敌人初始移动速度。
- 新增编辑器菜单，可快速创建关卡管理器和敌人出生点测试对象。

## 目前已实现功能

- Unity 2020.3.14f1 2D 项目基础脚本结构。
- 新版 Input System 键盘输入读取。
- M18 地面玩家：
  - 左右移动，并限制在当前摄像机视口范围内。
  - `X` 键跳跃，跳跃高度、滞空时间、下落速度可配置。
  - `Z` 键射击，子弹速度、生命周期、射速可配置。
  - 默认单路子弹，`↑ / ↓` 调整发射角度。
  - 支持通过 `AddBulletWay()` 增加额外弹路。
- M18 子弹：
  - 按方向飞行，生命周期结束后销毁。
- 基础敌人：
  - 可配置外观、血量、碰撞盒、初始左移速度。
  - 可配置射击速度、间隔、随机间隔、角度和子弹外观。
  - 被 M18 子弹命中后扣血，基础血量为 1 时被击毁。
- 背景块：
  - 支持在 Inspector 替换 Sprite。
  - 默认作为长条关卡背景放置，不参与关卡逻辑。
  - 背景创建菜单会沿 X 正方向铺设多块占位背景。
  - 可选开启对象自移动，用于临时演出。
- 关卡卷轴：
  - 关卡按 X 正方向延展。
  - 摄像机由关卡管理器匀速推进，可配置起点、终点和速度。
  - 玩家活动范围随摄像机移动。
- 敌人出生点：
  - 按摄像机右边缘触发。
  - 可配置敌人 Prefab、数量、队形偏移、生成间隔和初始移动速度覆盖。
  - 可组合多个配置条目形成简单敌人组。
- 编辑器测试菜单：
  - 创建 M18 玩家。
  - 创建基础敌人。
  - 创建背景块组。
  - 创建关卡卷轴管理器。
  - 创建敌人出生点。

## 版本记录

### v0.5.0

- 新增关卡卷轴管理器和敌人出生点。
- 背景卷轴逻辑改为摄像机推进式。
- 敌人基础类支持出生点覆盖初始移动速度。

### v0.4.0

- M18 射击改为默认单路。
- 支持上下方向键调整子弹角度。
- 支持道具接口增加额外弹路。
- 跳跃键改为 `X`。

### v0.3.0

- 实现可替换背景占位块。
- 实现基础背景层创建菜单。
- 初步支持远景、中景、地面背景配置。

### v0.2.0

- 实现 M18 玩家基础移动、跳跃、射击、屏幕范围限制。
- 使用占位图形，便于后续替换正式美术。

### v0.1.0

- 建立项目基础脚本结构。
- 新增基础敌人、敌人子弹、玩家子弹等核心测试脚本。
